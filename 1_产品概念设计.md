### 4.2 技术架构决策 | Architectural Decisions

**服务依赖注入模式 (Dependency Injection):**
```python
# ✅ 正确：通过 Depends() 注入
async def search_cases(
    courtlistener: CourtListenerClient = Depends(get_courtlistener)
):
    result = await courtlistener.search_opinions(...)

# ❌ 错误：在函数内调用全局获取函数
async def search_cases(...):
    client = await get_courtlistener(request)  # 不允许
```

**为什么使用 Depends()？**
- 便于测试：可以用 mock 对象替换真实服务
- 生命周期管理：服务在请求结束时正确清理
- 显式依赖：所有依赖关系清晰可见

**数据流架构:**
```
Search.tsx
    ↓ axios.get('/api/cases/search?q=...')
FastAPI: search_cases()
    ↓
CourtListenerClient.search_opinions()
    ↓ HTTP GET https://www.courtlistener.com/api/rest/v3/search/...
    ↓
SearchResponse { total, page, results[] }
    ↓
React Query 缓存
    ↓
UI 渲染
```
